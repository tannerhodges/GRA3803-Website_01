About Spry 1.4 and Dreamweaver

Spry 1.4 is the version of Spry thatâ€™s included with Dreamweaver CS3, so this version of the documentation (Spry 1.6) includes Spry features not available in Dreamweaver CS3. Additionally, the Spry 1.6 documentation does not discuss the visual tools that help you build Spry pages in Dreamweaver CS3. For information on how to use Spry tools in Dreamweaver, see Dreamweaver Help.

To consult the Spry 1.4 Developer Guide for use with Dreamweaver CS3, select Help > Spry Framework Help in Dreamweaver, or visit www.adobe.com/go/learn_dw_spryframework. The Spry 1.4 Developer Guide contains enhanced information on Spry functionality, beyond the visual authoring information that the Dreamweaver CS3 help system provides.

Versions of the Spry Developer Guide produced later than Spry 1.4 might not always be compatible with the Spry assets included in Dreamweaver. If you are a Dreamweaver CS3 user and want to upgrade to more current versions of the Spry framework, use the Spry Updater extension for Dreamweaver to download the latest Spry assets from Adobe Labs. If you do so, be sure that you are using the latest version of the documentation as well.
