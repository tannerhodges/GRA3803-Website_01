Change required status of radio buttons

By default, Validation Radio Button widgets require the user to make a selection before submitting the form. You can, however, make selections optional for the user.

 To change the required status of a Radio Button, add the isRequired property to the constructor and set its value to false, as follows: 
<script type="text/javascript">
	var spryRadio1 = new Spry.Widget.ValidationRadio("RadioWidget", {isRequired:false});
</script>
