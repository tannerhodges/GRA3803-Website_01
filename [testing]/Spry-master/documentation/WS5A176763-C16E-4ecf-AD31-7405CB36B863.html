Change required status of a Confirm widget

By default, Validation Confirm widgets require user input when published on a web page. You can, however, make the completion of text fields optional for the user.

 To change the required status of a widget, add the isRequired property to the constructor and set its value to false, as follows: 
<script type="text/javascript">
	var ConfirmWidgetObject = new Spry.Widget.ValidationConfirm("ConfirmWidget", "password1", {isRequired:false});
</script>
